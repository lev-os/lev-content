# Workflow and Lifecycle

Lev work follows a lifecycle progression from discovery to completion.

## Planning Phase

1. Discover context and constraints
2. Align with north star and boundaries
3. Research current state and prior art
4. Propose solution direction
5. Crystallize into executable scope

## Execution Phase

1. Execute against scoped tasks
2. Validate behavior and outputs
3. Emit handoff/status artifacts
4. Record learnings and close the loop

## Artifact Expectations

- Tracker work lives in BD
- Public docs pages live in `community/lev-content/docs/public`
- Internal specs/design remain in internal docs
