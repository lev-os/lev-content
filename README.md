# lev-content

Content repository for Lev ecosystem materials consumed by docs sites and tooling.

## Structure

- `docs/`: User-facing and source markdown content.
- `skills/`: Published skill docs and references intended for external consumption.
- `prompts/`: Prompt templates and workflow prompts.
- `assets/`: Shared static assets for docs/sites.

## Notes

This repo is intended to be mounted as a git submodule in downstream monorepos (for example `community/lev-content` in `leviathan`).
