## Key Concepts

### Code Visualization
- Don't look at lines of code — look at a **code map**: symbols, logic
- Live representation of everything happening in an application
- Agent tracing live — real-time representation

### 3D Mind Maps
- Graph visualization of current state and target state
- Expand/collapse in 3D space
- Navigate through time (past/future graph states)

### Interaction
- Hand gestures for interrupt/pause
- Scrolling text at controlled rates
- Same entities as mobile/desktop — just bigger and better in VR
- Research visible in VR space — interrupt or pause with gestures

### Prototyping Path
```
CLI (now) → 2D Desktop Dashboard → 3D VR Environment
```
- Can prototype on desktop first
- Unity or equivalent for screens + 3D space
- Need graphs, 3D mind map, graph visualization
- First: 2D representation, then extend to VR
